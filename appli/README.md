# Application web simple (etape 2)

Ce dossier contient une application web minimale en Python/Flask.

## Prerequis

- Python 3.10+

## Installation

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Lancement

```bash
python app.py
```

L'application est accessible sur `http://localhost:5000/`.
